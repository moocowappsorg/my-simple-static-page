# Simple Static Page

A clean, responsive static HTML page with inline CSS.

## Features

- Simple, modern design
- Fully responsive layout
- Inline CSS (no external dependencies)
- Easy to deploy to any static hosting service

## Usage

Simply open `index.html` in any web browser.

## Deployment

This project is ready to deploy to services like:
- Render
- Netlify
- Vercel
- GitHub Pages

Just point the deployment service to this repository.
